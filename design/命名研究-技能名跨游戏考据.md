## 二、与 Jonasun 调色盘的合成(命名法 v2)

**统一判据(Jonasun 2026-07-11)**:抽象但直观、贴切;**多套逻辑混用,同逻辑扎堆=无聊**。

本作调色盘(七逻辑,对应 Arena 验证):
1. **概念称谓**(=Arena②):第五个答案 / 不化的雪 / 深渊的注视 / 盛开的刀
2. **诗意二字**(本作特色,Arena 无对应——中文优势):乘风 / 织衣 / 余烬 / 回声 / 饮血 / 出鞘
3. **直观动态**(=Arena①):连续击鼓 / 狂野生长
4. **态度熟语**(=Arena④变体):成王败寇 / 趁火打劫 / 寅吃卯粮 / 落袋为安 / 先斩后奏 / 钝刀也是刀
5. **誓言体**(=Arena④):只练这一锤
6. **口号感叹**(=Arena④):手感来了
7. **称谓身份**(=Arena⑤):破壁人 / 赌徒之门
8. **系列家族**(=Arena⑦):雪系连作(早到的雪 / 不化的雪)——未来可扩
9. **俏皮考试域梗**(=Arena③,**候补卡专用金矿**):交卷 / 附加题 / 超纲了 / 标准答案 / 抢答器 / 借同桌一支笔——考试域双关必须机制兑现

技能名(非强化)~~独立一律:二字动词招式~~(2026-07-11 作废:掐表/划叉/回血/撑伞被判难听——纯工具动词没有气质)。**技能名与强化名同调色盘**:结账(态度)/闭关(诗意武侠)/时间到(考试域:监考喊停)/排除法(考试域:做题第一技);保留的旧名恰好都自带气质(撕卷/起浪/雪崩/点名/举手/一闪/擂鼓)。觉醒名重一档且尽量与本体成对:闭关→出关,结账→回本(金钱族),时间到→偷秒。

## 四、给未来卡的命名流程(并入宪法五关的语域关)

1. 先判卡的气质:赌卡→态度;成长→直观动态或诗意;控制→概念或诗意;考试道具→俏皮域内梗
2. 在调色盘里选**当前池最稀缺**的逻辑(看本文三节分布表)
3. 过三测:抽象但直观?念出来有劲?域内双关兑现了吗?
4. 撞名黑名单 + 与同池名字的逻辑重复度检查

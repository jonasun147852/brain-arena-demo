# 强化生成系统 v1 · 2026-07-10

> **这是核心玩法的宪法。** 每设计一张新卡/新机制,按本文五关走一遍;每次改卡后跑 `node tools/extract-pool.mjs` 重新提取数据、重发布面板。可视化面板:design/pool-dashboard.html(Artifact 在线版由 Claude 维护)。

## 用法(生成一张新卡的流程)

1. **动词关**:它占据哪个没人占的动词象限?(冻结=停表/减速=慢表/点名=倒表/举手=停自己的表)换皮不收。
2. **基因关**:若是赌卡,过赌感北极星三要素;若是求助技,过总闸门;若是觉醒,改行为不加倍率。
3. **语域关**:名字过二字动词核(赌卡族许四字熟语),desc 过小学生朗读,flavor 一句狠话,不撞黑名单。
4. **数值关**:标功率价格,对照红线表;上 harness 三种子打靶(强 3/3、均 2-3/3、弱死中后段)。
5. **池健康关**:入哪个池哪一轮?流派/稀有度/槽位配额还平吗?(看面板池健康区)

## 二、设计偏好(不可违反)

**赌感北极星**(Jonasun 2026-07-10 钦定,见 agent 记忆):
1. **可选入**——玩家自己按赌注按钮,不是被动 RNG;
2. **押自己**——赌手气/手速/判断,不是纯骰子;
3. **会背叛但输端有定价**——卡壳减半不归零、押空只没收本金。
在册赌感族:赌徒之门/成王败寇/趁火打劫/寅吃卯粮/一鼓作气/落袋为安/先斩后奏。

**语域四律(2026-07-11 二修:命名调色盘 v2,详见 design/命名研究-技能名跨游戏考据.md)**:①技能名=**机制直达优先**(2026-07-11 四修,Jonasun 亲改:闭关→举盾/结账→重击——技能名必须一听就知道干什么,弯一道的比喻判死;双关可以但机制必须直达[时间到/排除法留任];觉醒名与本体成对[举盾→盾击]);②**强化名=海克斯命名法**——概念的完整称谓:完整名词短语+限定词确指(第五个答案/不化的雪/借来的风),命名现象不命名功能;压缩缩写与说教成语=人机感,态度型四字熟语(成王败寇族)保留;③desc=**游戏术语体**(造成/定身/冻结/燃烧/持续伤害/额外攻击/回复/格挡/倒转),禁口语禁说明书;④~~flavor 一句狠话~~(2026-07-11 Jonasun 裁决:卡面**只留机制描述**——副题 idline 与狠话引言全部下卡,不再显示;fl/idline 数据字段保留但任何界面不得渲染。名字本身就是挑出来的,不需要说明文陪衬)。

**觉醒范式**:资格=放满 3 次(○○○可视);出现=后两抽独立 roll(45%/65%)不保底;pity 跨局累进出现即清;升级改行为不加倍率(观察位:再闪+33%、连擂)。

**求助技总闸门**:求助的题奖励减半/归零 → 求助永远净亏能量,买保命不买收益。

**局间方差(鸿运局)**:骰局不骰手 6%;只在中奖时宣告;基线一毛不拔;弃局钓不到;无 pity 无假近失。

## 三、数值平衡框架

**四道帽**:攻击加法共享帽 16 · 溅射帽 0.9 · 过载 5 层(渊眼 8)· 溢出结转 ≤下阶段预算 50%。
**经济**:费用地板 轻25/重50;折扣链 g02(自地板30/70)→h12(−20)→全局地板;恒价卡(砸卷40)不进折扣链;能量上限 150;答题是唯一能量源。
**控制**:充能护栏——同一充能周期第二次干预效果减半(清零→半清,倒流→半速),enemyAttack 复位;一条规则管所有现在与未来的控制。
**红线**:禁指数倍率;模拟单发均值 ≤400(双窗叠乘的蹲窗大奖除外);鸿运局 ≤900;敌方每关出手次数 ≥基线 60%。
**对局骨架(v5.4)**:发牌 0/30/75/150/210s;血池 [260,880,1580];ATK [7,7,11];充能 [10,8.5,7.5]s;HP100 回50;狂暴 26 题或 140s;轮色权重 银65/金35→…→金40/彩60(鸿运表无银)。
**打靶流程**:改数值 → scratchpad harness_v2 三种子 → 目标带:强 3/3 约230s 提前收官 / 均 2-3/3 约250-290s / 弱死中后段 / 囤 2-3/3;超线处置次序:权重 → 帽 → 最后才动血池。

# 音频节奏设计 · 心理学依据与实现规格(v5.30)

> 触发:Jonasun 2026-07-12「背景音乐完全没有节奏感,太阴沉了…base on 心理学…让人欲罢不能」。
> 结论先行:**120 BPM 四踩底 + 连击垂直分层 + 大招音乐闪避**。

## 一、研究结论(带出处)

1. **多巴胺三件套=期待+不确定+即时反馈**;声音稳定伴随成功时刻会被操作性条件反射绑定为满足信号(答对音必须可靠、递进)。老虎机套路:音高/节奏渐升+战略停顿放大期待。
   - 来源:[IJRPR 游戏背景音乐心理效应](https://ijrpr.com/uploads/V5ISSUE7/IJRPR31637.pdf) / [声音与玩家情感 meta 分析](https://www.researchgate.net/publication/395103967) / [老虎机音频](https://www.orbitalaffairs.com/how-music-supercharges-slot-game-excitement-and-player-engagement/)
2. **BPM 科学**:>120 BPM 可靠提升唤醒度;**认知任务(记忆/处理速度)在 ~120 BPM 最优**,慢速音乐显著拖垮处理速度,165 BPM 只到平庸——答题是认知任务,**基础 120,决战关封顶 134,不越 140**。旧版无节拍暗涌≈慢速=最差解。
   - 来源:[Frontiers 音乐速度与认知处理速度](https://www.frontiersin.org/journals/psychology/articles/10.3389/fpsyg.2023.998460/full) / [120bpm 记忆最优](https://www.researchgate.net/publication/235338088) / [pre-task tempo 唤醒](https://journals.sagepub.com/doi/10.1177/20592043231174388)
3. **垂直分层=行业标准**:整曲拆 stems,按状态开关;格斗游戏先例:连续命中逐层加、挨打减层。
   - 来源:[竞技游戏自适应音乐](https://www.gamedeveloper.com/audio/adaptive-music-in-competitive-games) / [垂直分层 vs 水平重排](https://www.thegameaudioco.com/making-your-game-s-music-more-dynamic-vertical-layering-vs-horizontal-resequencing) / [GDC 2021 纯垂直分层](https://www.gamedeveloper.com/game-platforms/pure-vertical-layering-for-game-music-composers-from-spyder-to-sackboy-gdc-2021-)
4. **打击感音频**:声音=体验的一半;单一音效不专业,行业标准三层叠(瞬态/材质/尾);拳感靠瞬态控制不靠音量;配合顿帧/震屏。
   - 来源:[Game feel on the web](https://valdemird.com/blog/game-feel-on-the-web/) / [GameAnalytics juice](https://www.gameanalytics.com/blog/squeezing-more-juice-out-of-your-game-design) / [打击音效分层](https://add.app/sound-effects/punching-sound-effects/)

## 二、实现规格(v5.30 音乐引擎 v3,纯 WebAudio 零素材)

- **骨架**:16 步进音序器,lookahead 调度(25ms tick / 0.12s 前瞻,ctx.currentTime 走表)。
- **BPM 每关**:[120, 126, 134] —— 唤醒递增但不出认知最优带。
- **乐器(全合成)**:kick(sine 150→45 + 点击瞬态)/ hat(高通噪声 6k 闭镲,开镲长尾)/ snare(带通噪声 1.8k)/ bass(五声根音方波低通,八分句)/ arp(三角波五声琶音,两小节短语)。
- **垂直分层 × 连击档**(comboTier 0-3 实时驱动,每步读状态):
  - L0 常驻:kick 四踩 + bass 八分(有节奏感的底,即使 0 连击)
  - L1(3 连):反拍闭镲
  - L2(6 连):五声琶音旋律入场
  - L3(9 连):十六分开镲抖 + 琶音上八度 —— 心流=满编制
  - 狂暴:加 snare 4/12 + 镲加倍(压迫)
  - 发牌/面板(时间冻结):只留 bass 半音量(仪式性抽走节拍)
- **音乐闪避(sidechain)**:重技/击破/受击瞬间音乐总线压到 35% 再 0.25s 恢复——给打击音让出瞬态,拳拳到肉。
- **总线**:music bus(gain ~0.13)→ master 压缩器;静音沿用 🔊 键;对外函数名不变(startAmbient/stopAmbient)。

## 三、判据(回访验收用)
- 开局 10 秒内脚会不会跟着点 = 节奏感及格线
- 连击 6+ 时音乐是否明显"起来了" = 分层被感知
- 放大招那一下音乐是否"让位" = 闪避起效
- 一局打完摘耳机是否觉得吵 = 音量纪律(音乐永远垫在音效下面)

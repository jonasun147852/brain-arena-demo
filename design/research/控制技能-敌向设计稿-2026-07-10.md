# 敌向控制技能设计 v1 — 三候选(收卷 / 没收 / 点名)

设计原则:把"打断对面答题/白屏/题目消失"翻译成对守门者充能条的三个**新动词**——**提前引爆**(收卷)、**偷走变现**(没收)、**倒流**(点名)。现有动词空间(震慑=清零、冻结=暂停、减速=×0.5、拨挡=一次性击退、硬直=暂停+易伤)全部是"把刀推迟";三个新动词分别是"把刀拉早"、"把条当战利品"、"让条倒着走",机制上零重叠。全部吃充能护栏(见文末统一护栏改造)。

---

## 候选一:收卷(轻技槽,银,心系,40⚡)

**名**:收卷 ·「时间到,笔放下。」 idline:出刀时间,你说了算
**desc**:轻技能换成:现在就收它的卷——这刀**提前落地**,力道只剩它**攒了几成**

**机制**:强制守门者立刻出手,伤害 = `round(E.atk × nextAtkMult × f)`,`f = max(0.15, E.charge/(E.chargeNeed*1000))`(地板 15%:空条收卷也要挨一下轻的)。出手后充能归零(自然消耗),进入新周期。**护栏约束:强制出手不重置 `stunCycleGuard`**——堵死"收卷洗周期→震慑再次全清"的洗护栏链。施放门槛:`!isControlled()` 且非审判题。

**为什么新**:全部现有控制都是把攻击推向未来;收卷是唯一把攻击**拉到现在**并缩水的动词。玩家从"躲时间"变成"定时间"——难题前、狂暴前、拨挡窗撞在坏时机时,主动选一个便宜时刻把这刀吃掉。考场双关直白:监考的变成你,「收卷!」小学生秒懂。

**演出**:守门者面孔立刻加 `lunge` class 扑出一记软刀,floatText「📄 收卷!」在敌侧,你侧掉一个小伤害数字,充能条瞬间抽干(width 动画归零)。它的 taunt 被截断感天然成立。

**觉醒:判卷** ·「这份卷子,零分。」——收卷改判:你挨的这一下,**双倍记它头上**(`dealDamage(2×你受到的伤害,'判卷',gold,raw)`)。行为增量不加倍率,符合觉醒范式。

**挂点**:`enemyAttack(mult=1, keepGuard=false)` 加两参:`mult` 乘进 1216 行 `atkDmg`;`keepGuard=true` 时跳过 1203 行 `E.stunCycleGuard=false`。releaseSmall 加 `sk.key==='collect'` 分支。

## 候选二:没收(轻技槽,银,脑系,35⚡)

**名**:没收 ·「上课玩刀?没收了。」 idline:它攒的家当,归你
**desc**:轻技能换成:没收它 **40%** 充能,偷来的全变你的**能量**

**机制**:`cut = E.charge × 0.4 × m`(m=护栏乘数,见文末),`E.charge -= cut`;返还能量 `gain = min(20, round(cut/1000 × 8))`(每偷 1 秒 = 8⚡)。满条时(6.5s 档)最大偷 2.6s ≈ 20⚡,净费 15⚡ + 白赚 40% 延时。赌徒之门卡壳时 m 再减半。

**为什么新**:现有控制只"否定时间",没收把敌人的进度**变成你的资源**——诞生一个贪心小游戏:条越肥偷得越多,但等条肥=刀越近。风险定价由玩家自己出,这是震慑/冻结给不了的决策张力。与 落井下石(x03)、熟能生巧(g02)形成经济流协同。「没收」是教室里最有权力感的词。

**演出**:敌方充能条被"咔"地掰断一截(width 跳变+断口闪光),floatText 敌侧「📎 没收!」,一串 `+N⚡` 飘回你侧能量槽——偷窃感可视化。

**觉醒:充公** ·「连你下一刀,也充公。」——没收升级:偷完充能,再没收它下一刀的一半:`E.nextAtkMult = min(E.nextAtkMult, 0.5)`(一次性)。enemyAttack 1217 行加 `else if(nextAtkMult<1)` 浮字「×½!」。纯行为增量,复用现有 nextAtkMult 挂点(千钧一发的 ×2 走同一通道,互为镜像)。

**挂点**:releaseSmall 加 `sk.key==='confiscate'` 分支;能量走 `P.energy = min(K.ENERGY_MAX, ...)`。

## 候选三:点名(重技槽,金,心系,100⚡)

**名**:点名 ·「下一个,你来答。」 idline:叫到名字,条往回走
**desc**:重技能换成:攻击 **×3**,点它的名——充能**倒着走** 3.5 秒

**机制**:heavySkill 表加 `{key:'rollcall', base:3, reverse:3500}`。施放时 `m = tamperGuard()`,`E.reversedUntil = now+3500; E.reverseRate = m`(乘数施放瞬间锁定,不吃自己标记的护栏)。combatTick 在受控检查**之前**加分支:`if(now < E.reversedUntil){ E.charge = max(0, E.charge - dt × E.reverseRate); E.parryWindow = false; updateSceneUI(); return; }`。`isControlled()` 加 `t<E.reversedUntil`(喂给 x03/先发制人);2441 行暂停平移加 `E.reversedUntil += dt`。

**平衡账**:倒流 1 秒 = 停 1 秒 + 退 1 秒 = 2 秒否定值 → 3.5s 倒流 ≈ 7s 否定,对标掐表(100⚡,×4 + 5s 冻结);故伤害压到 ×3。护栏命中时倒速减半(≈5.25s 否定),同周期二连控自动衰减。与掐表叠放:冻结分支先 return,倒流分支放在其前所以照常抽条——200⚡ 的豪华连招,允许,但第二次 tamper 已被护栏减半。

**为什么新**:负速率是充能系统里唯一没人占的象限——冻结停表、减速慢表、震慑摔表,点名让表**倒着走**。演出即机制:条肉眼可见地往回缩,压迫感反转。课堂双关最狠:守门者被老师点名,轮到**它**站起来答题——"打断对面答题"的最字面翻译。

**演出**:banner「🫵 点名!下一个,它答!」,充能条 fill 换紫色+反向流光 CSS class,状态徽章 `🫵点名`(2358 行旁),敌核加轻微 shiver 动画。

**觉醒:罚站** ·「答不上来,就站好。」——点名升级:倒流期间条被抽干到 0,它**罚站 2 秒**(`E.frozenUntil=+2000; E.staggeredUntil=+2000`,站着挨打更疼,走既有硬直通路)。一次性触发标记 `E._rollcallPunished`。条件型行为增量:奖励"点名点在肥条上"的时机功。

## 统一护栏改造(三卡共用,一处代码)

```js
function tamperGuard(){          // 充能干预护栏推广:同周期第二次任何干预,效果减半
  const m = E.stunCycleGuard ? 0.5 : 1;
  E.stunCycleGuard = true;
  return m;
}
```
`clearCharge()`(1235)语义不变(全清→半清 即 m=1→0.5 的特例),可改写复用。没收的偷取量、点名的倒速都乘 m;收卷不清条但**保留**护栏标记。一条规则管五个动词,控制链数学上必衰减。
